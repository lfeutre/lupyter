## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {lupyter, ".*",
      {git, "git@github.com:oubiwann/lupyter.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
