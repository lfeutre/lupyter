## Dependencies

You will need to have the following installed:

* Erlang
* Python
* ``pip``
* ZeroMQ

## Installation

```
$ git clone https://github.com/oubiwann/lupyter.git
$ cd lupyter
$ make
```


## Usage


For terminal use:

```
$ make console
```

For use in a notebook:

```
$ make notebook
```
